# PyPNM-WebUI Wrapper

PyPNM-WebUI is now a wrapper repository.

This repo keeps compatibility entrypoints for teams that still begin from
`PyPNM-WebUI`, but all active runtime/frontend development is now maintained in
`PyPNM-CMTS-WebUI`.

Use this repository for:

- wrapper install and uninstall commands
- migration notes to unified mode
- compatibility documentation

Use the unified repository for active WebUI development:

- https://github.com/PyPNMApps/PyPNM-CMTS-WebUI
